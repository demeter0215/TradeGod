# 做T策略技能

> ⚠️ **风险提示**：本文档仅供学习研究使用，不构成任何投资建议。做T策略需要丰富的交易经验，新手谨慎尝试。

## 一、策略原理

### 1.1 什么是做T

在A股**T+1交易制度**下，当日买入的股票当日不能卖出。做T是通过持有底仓，在日内进行**先卖后买（反T）**或**先买后卖（正T）**的操作，实现变相T+0，以降低持仓成本。

```
┌─────────────────────────────────────────────────────────────────┐
│                      做T基本原理                                 │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   正T（先买后卖）：                                              │
│   持有底仓 ──> 低价买入 ──> 高价卖出同等数量 ──> 持仓不变        │
│                                   ↓                             │
│                              赚取差价                           │
│                              降低成本                           │
│                                                                 │
│   反T（先卖后买）：                                              │
│   持有底仓 ──> 高价卖出 ──> 低价买回同等数量 ──> 持仓不变        │
│                                   ↓                             │
│                              赚取差价                           │
│                              降低成本                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 做T的核心价值

| 价值 | 说明 |
|------|------|
| **降低成本** | 通过日内波动摊薄持仓成本 |
| **锁定利润** | 反弹时卖出，回落时买回 |
| **控制回撤** | 下跌时减仓，止跌后买回 |
| **提升效率** | 利用闲置资金创造收益 |

### 1.3 做T的前提条件

1. **必须有底仓**：做T建立在已有持仓基础上
2. **股票波动大**：日均振幅最好在3%以上
3. **流动性好**：日成交额最好大于1亿
4. **趋势明确**：上升或震荡趋势，避免单边下跌

## 二、日内做T技巧

### 2.1 正T vs 反T

| 类型 | 操作顺序 | 适用场景 | 难度 | 风险 |
|------|----------|----------|------|------|
| **正T** | 先买后卖 | 预期上涨，低点买入 | 低 | 买入后可能继续跌 |
| **反T** | 先卖后买 | 预期回调，高点卖出 | 高 | 卖出后可能继续上涨 |

**建议**：新手从正T开始，熟练后再做反T

### 2.2 日内分时做T要点

```
┌─────────────────────────────────────────────────────────────────┐
│                    日内分时做T技巧                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 确定基准线                                                  │
│      - 昨日收盘价                                                │
│      - 当日均线（黄线）                                          │
│      - 关键支撑/阻力位                                           │
│                                                                 │
│   ② 观察分时形态                                                │
│      - 放量上涨 → 正T机会                                        │
│      - 缩量回调 → 正T买点                                        │
│      - 放量滞涨 → 反T卖点                                        │
│      - 缩量反弹 → 反T机会                                        │
│                                                                 │
│   ③ 设置目标位                                                  │
│      - 正T：买入价 + 1%~2% = 卖出目标                          │
│      - 反T：卖出价 - 1%~2% = 买回目标                          │
│                                                                 │
│   ④ 严格执行                                                    │
│      - 达到目标立即执行                                          │
│      - 不贪婪、不犹豫                                            │
│      - 做错立即止损                                              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2.3 分时图经典形态

**适合正T的形态**：
- **W底**：分时双底，第二底高于第一底
- **放量突破**：突破分时均线且放量
- **缩量回调**：上涨后缩量回调至均线

**适合反T的形态**：
- **M头**：分时双顶，第二顶低于第一顶
- **放量滞涨**：创新高但量能萎缩
- **破位下跌**：跌破分时均线且放量

## 三、开盘前1小时交易规律（9:30-10:30）

### 3.1 早盘时段特点

```
┌─────────────────────────────────────────────────────────────────┐
│                    早盘三阶段                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   9:30-9:45  开盘博弈期                                          │
│   ├─ 特点：情绪波动大，成交量集中                                │
│   ├─ 机会：集合竞价错杀后的修复                                  │
│   └─ 风险：追高容易被套                                          │
│                                                                 │
│   9:45-10:00 趋势确立期                                          │
│   ├─ 特点：初步方向确立                                          │
│   ├─ 机会：顺势操作                                              │
│   └─ 建议：观察为主，等待明确信号                                │
│                                                                 │
│   10:00-10:30 消化调整期                                         │
│   ├─ 特点：早盘趋势确认或反转                                    │
│   ├─ 机会：日内第一个小低点                                      │
│   └─ 策略：正T较好时机                                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 早盘做T策略

**策略一：集合竞价观察**
- 观察高开/低开幅度（>3%需谨慎）
- 观察竞价量能（是否异常放大）
- 观察涨跌停家数（判断市场情绪）

**策略二：开盘15分钟不操作**
- 等待情绪释放
- 观察主力资金动向
- 确定日内趋势方向

**策略三：10点后的正T机会**
- 若早盘放量上涨后缩量回调
- 回调幅度在1%-2%之间
- 在分时均线附近企稳

### 3.3 早盘禁忌

| 禁忌 | 原因 | 后果 |
|------|------|------|
| 开盘追涨 | 情绪高点 | 当日被套 |
| 开盘割肉 | 情绪低点 | 卖在最低点 |
| 频繁操作 | 波动大易出错 | 成本越做越高 |
| 全仓做T | 无回旋余地 | 一旦做错被动 |

## 四、收盘前半小时交易规律（14:30-15:00）

### 4.1 尾盘时段特点

```
┌─────────────────────────────────────────────────────────────────┐
│                    尾盘两阶段                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   14:30-14:50 尾盘躁动期                                         │
│   ├─ 特点：资金抢筹或出逃                                        │
│   ├─ 异动：大幅拉升或跳水                                        │
│   └─ 原因：消息预期、资金调仓                                    │
│                                                                 │
│   14:50-15:00 收盘定型期                                         │
│   ├─ 特点：价格趋于稳定                                          │
│   ├─ 异动：竞价异动                                              │
│   └─ 策略：确定隔夜仓位                                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 尾盘做T策略

**尾盘正T（低吸）**：
- 适用：当日下跌，尾盘企稳
- 信号：14:30后不再创新低
- 目标：博弈次日反弹
- 仓位：不超过底仓的50%

**尾盘反T（高抛）**：
- 适用：当日大涨，尾盘滞涨
- 信号：14:30后放量滞涨
- 目标：锁定当日利润
- 买回：次日低点买回

### 4.3 尾盘注意事项

1. **不赌消息**：尾盘异动不盲目跟风
2. **控制仓位**：尾盘做T仓位要轻
3. **留有余地**：保留部分资金应对次日
4. **观察竞价**：最后3分钟竞价有参考意义

## 五、隔日做T策略

### 5.1 隔日做T原理

当日内波动不足以做T，或当日走势不及预期时，可将做T周期延长至隔日。

```
┌─────────────────────────────────────────────────────────────────┐
│                    隔日做T模式                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   模式一：下跌补仓型                                             │
│   Day1：持仓下跌，未达止损位                                     │
│   Day2：开盘继续下跌，补仓                                       │
│   Day3：反弹，卖出补仓部分                                       │
│   结果：成本降低，持仓不变                                        │
│                                                                 │
│   模式二：反弹减仓型                                             │
│   Day1：持仓反弹，遇阻力位                                       │
│   Day2：冲高减仓                                                 │
│   Day3：回调，买回减仓部分                                       │
│   结果：赚取差价，降低成本                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 隔日做T适用场景

| 场景 | 操作 | 逻辑 |
|------|------|------|
| 突发利空 | 隔日低点补仓 | 错杀修复预期 |
| 技术回调 | 隔日低点补仓 | 趋势未改 |
| 快速反弹 | 隔日高点减仓 | 短线获利了结 |
| 遇到阻力 | 隔日减仓观望 | 规避调整风险 |

## 六、量价关系在做T中的应用

### 6.1 核心量价关系

```
┌─────────────────────────────────────────────────────────────────┐
│                    做T量价法则                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 量增价涨 ──> 正T机会（趋势确认，可追）                      │
│                                                                 │
│   ② 量缩价跌 ──> 正T机会（抛压减轻，将企稳）                  │
│                                                                 │
│   ③ 量增价平 ──> 反T机会（滞涨信号，将回调）                  │
│                                                                 │
│   ④ 量缩价涨 ──> 反T机会（无量上涨，不可持续）                │
│                                                                 │
│   ⑤ 量增价跌 ──> 观望（恐慌抛售，等企稳）                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 6.2 分时量能观察

**正T量能信号**：
- 回调时成交量明显萎缩
- 企稳时出现小阳线+放量
- 突破分时均线时放量

**反T量能信号**：
- 上涨时成交量递减
- 高位出现长上影线+放量
- 跌破分时均线时放量

## 七、适用场景

### 7.1 适合做T的情况

| 情况 | 说明 | 预期收益 |
|------|------|----------|
| 震荡市 | 股价区间内波动 | 降低持仓成本 |
| 持仓被套 | 不想割肉 | 逐步解套 |
| 趋势上升 | 日线级别上涨 | 增厚收益 |
| 底仓充足 | 有做T基础 | 提升资金效率 |

### 7.2 不适合做T的情况

| 情况 | 风险 | 建议 |
|------|------|------|
| 单边下跌 | 越做越套 | 止损观望 |
| 单边上涨 | 容易卖飞 | 持股不动 |
| 波动极小 | 收益不够手续费 | 放弃做T |
| 重大消息前 | 不确定性大 | 等明朗后再操作 |

## 八、风险控制要点

### 8.1 做T铁律

1. **不满仓做T**：始终保持部分底仓
2. **不逆势做T**：下跌趋势不做正T
3. **严格止损**：做错立即纠正
4. **控制频率**：每日最多做T一次
5. **先算手续费**：确保差价覆盖成本

### 8.2 常见错误

| 错误 | 后果 | 纠正方法 |
|------|------|----------|
| 正T买在半山腰 | 仓位加重被套 | 等企稳再买入 |
| 反T卖飞 | 错失涨幅 | 保留底仓，分批卖 |
| 贪心 | 盈利变亏损 | 设定目标，严格执行 |
| 赌气 | 越错越加仓 | 认错止损，冷静后再操作 |

## 九、实战案例参考

### 案例1：成功正T

**背景**：持有某科技股10000股，成本20元

**Day1操作**：
- 早盘低开至19.5元（-2.5%）
- 观察10分钟，企稳后买入3000股
- 午后拉升至20.2元
- 卖出3000股

**结果**：
- 赚取差价：(20.2 - 19.5) × 3000 = 2100元
- 持仓成本降低：2100 ÷ 10000 = 0.21元
- 新成本：19.79元

### 案例2：失败反T

**背景**：持有某股5000股，成本30元

**Day1操作**：
- 早盘冲高至31元（+3.3%）
- 认为会回调，卖出2000股
- 结果继续上涨，收盘32元

**Day2操作**：
- 开盘32.5元，犹豫未买回
- 继续上涨，被迫在33元买回

**教训**：
- 反T难度高，容易卖飞
- 强势股不轻言做空
- 建议新手少做反T
