# 激进搏杀策略

> ⚠️ **重要风险提示**：
> - 本策略风险极高，可能导致本金大幅亏损
> - 仅适合有经验、有风险承受能力的投资者
> - 单笔交易资金不得超过总资金的10%
> - 新手请勿尝试，建议先在模拟盘验证

## 一、策略原理

### 1.1 什么是激进搏杀策略

激进搏杀策略是在A股市场中，利用**涨停板制度**和**情绪周期**，以小资金博取短期高收益的短线交易策略。

```
┌─────────────────────────────────────────────────────────────────┐
│                  激进搏杀策略核心逻辑                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   本质：博弈市场情绪和资金接力                                   │
│                                                                 │
│   核心假设：                                                     │
│   ① 涨停板会吸引市场关注                                        │
│   ② 强势股有惯性上涨动力                                        │
│   ③ 情绪周期有可预测的规律                                      │
│   ④ 资金会追逐赚钱效应                                          │
│                                                                 │
│   盈利模式：                                                     │
│   今日买入 ──> 明日高开/涨停 ──> 获利了结                      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 策略适用前提

**必须同时满足**：
1. 市场有明确热点和主线
2. 涨停家数 > 50家（情绪活跃）
3. 有连板龙头引领
4. 本人有丰富短线经验
5. 能承受单笔10%以上亏损

**绝对不做的情况**：
- 市场整体跌停家数 > 10家
- 高位龙头断板大跌
- 没有明确主线
- 自己状态不佳
- 连续亏损后（禁止报复性交易）

## 二、涨停板战法

### 2.1 首板战法

**定义**：某股票第一个涨停板的交易机会

**首板类型**：

```
┌─────────────────────────────────────────────────────────────────┐
│                      首板分类                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 突破首板                                                   │
│      - 突破长期平台或前高                                       │
│      - 量能放大，换手充分                                       │
│      - 质量最高，关注度高                                       │
│                                                                 │
│   ② 消息首板                                                   │
│      - 突发利好消息刺激                                         │
│      - 需判断消息持续性                                         │
│      - 一字板无法参与                                           │
│                                                                 │
│   ③ 跟风首板                                                   │
│      - 板块龙头已涨停                                           │
│      - 同板块补涨                                               │
│      - 持续性一般                                               │
│                                                                 │
│   ④ 超跌首板                                                   │
│      - 长期下跌后首板                                           │
│      - 可能是反弹也可能是反转                                   │
│      - 需结合大盘判断                                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**首板买入方法**：

| 方式 | 时机 | 风险 | 适合 |
|------|------|------|------|
| 打板 | 即将涨停时买入 | 高（可能炸板） | 确定性强的突破板 |
| 回封 | 涨停打开后回封 | 中等 | 换手充分的板 |
| 竞价 | 集合竞价买入 | 较高 | 隔夜利好预期 |

**首板选择标准**：
- 流通市值：50-200亿（太小流动性差，太大拉不动）
- 股价：10-30元（散户参与度高）
- 换手：3%-15%（充分但不过分）
- 封单：大于流通盘的1%
- 板块：当日热点前排

### 2.2 连板战法

**定义**：已经有一个或多个涨停板的股票继续参与

**连板类型**：

```
┌─────────────────────────────────────────────────────────────────┐
│                    连板梯队                                     │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   2板（一进二）                                                 │
│   ├─ 从首板中选出                                               │
│   ├─ 成功率约20-30%                                             │
│   └─ 竞争最激烈的环节                                           │
│                                                                 │
│   3板（二进三）                                                 │
│   ├─ 市场关注度显著提升                                         │
│   ├─ 开始成为板块龙头候选                                       │
│   └─ 成功可能成为大龙头                                         │
│                                                                 │
│   4板及以上                                                     │
│   ├─ 板块龙头或市场龙头                                         │
│   ├─ 需考虑监管风险                                             │
│   └─ 高位博弈，风险收益比下降                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**连板参与原则**：
1. **只做最强**：同身位只做最早涨停的
2. **看高做低**：有高位龙头护航，低位更安全
3. **换手优先**：一字板后开板承接更重要
4. **放弃末位**：梯队后排容易大面

### 2.3 反包战法

**定义**：强势股断板后，次日或几日内再次涨停，反包前高

**反包类型**：

| 类型 | 特征 | 胜率 | 买点 |
|------|------|------|------|
| 次日反包 | 断板次日直接涨停 | 高 | 打板确认 |
| 隔日反包 | 断板后调整一天再涨停 | 中 | 首板打板 |
| 平台反包 | 断板后横盘整理再突破 | 中高 | 突破打板 |

**反包买点**：
- 必须是市场核心人气股
- 断板当天不能是大阴线（>-7%）
- 反包日必须放量涨停
- 板块效应仍在

## 三、龙头战法

### 3.1 龙头的定义

**真龙头的特征**：

```
┌─────────────────────────────────────────────────────────────────┐
│                    真龙头特征                                    │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 高度最高                                                    │
│      - 同板块连板数最多                                         │
│      - 市场最高板之一                                           │
│                                                                 │
│   ② 带动性最强                                                  │
│      - 涨停后板块跟风                                           │
│      - 板块内个股涨停数增加                                     │
│                                                                 │
│   ③ 辨识度最高                                                  │
│      - 名字好记、有特点                                         │
│      - 股性好（历史多次涨停）                                   │
│                                                                 │
│   ④ 抗跌性最好                                                  │
│      - 调整时幅度最小                                           │
│      - 率先企稳反弹                                             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 龙头买点

**买点一：确认龙头的首个涨停**
- 2进3或3进4时确认龙头地位
- 打板买入

**买点二：龙头首次分歧**
- 涨停打开后回封
- 放量换手后回封是机会

**买点三：龙头首阴**
- 首次不涨停（但不是大阴线）
- 次日反包预期
- 高风险，新手不做

### 3.3 龙头卖点

**必须卖出的信号**：
1. 放量跌停
2. 尾盘炸板不回封
3. 板块全面退潮
4. 出现新龙头替代
5. 达到预期盈利目标

**分批卖出策略**：
- +10%：卖出30%
- +20%：卖出30%
- +30%以上：清仓或留底仓

## 四、情绪周期判断

### 4.1 情绪周期四阶段

```
┌─────────────────────────────────────────────────────────────────┐
│                   情绪周期循环                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│    ┌─────────┐      ┌─────────┐                                │
│    │  冰点   │──────>│  启动   │                                │
│    │  (空仓) │      │ (轻仓)  │                                │
│    └─────────┘      └────┬────┘                                │
│         ↑                │                                      │
│         │                ↓                                      │
│    ┌────┴────┐      ┌─────────┐                                │
│    │  退潮   │<─────│  高潮   │                                │
│    │ (减仓)  │      │ (重仓)  │                                │
│    └─────────┘      └─────────┘                                │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 各阶段特征与策略

| 阶段 | 市场情绪 | 涨停家数 | 最高板 | 策略 |
|------|----------|----------|--------|------|
| **冰点** | 恐慌 | <30家 | 3板以下 | 空仓观望 |
| **启动** | 试探 | 30-50家 | 3-4板 | 轻仓试错 |
| **高潮** | 亢奋 | >80家 | 6板以上 | 重仓龙头 |
| **退潮** | 谨慎 | 快速减少 | 断层 | 减仓离场 |

### 4.3 情绪判断指标

**量化指标**：
- 涨停家数/跌停家数比值
- 昨日涨停今日表现指数
- 炸板率（开板数/涨停数）
- 连板率（连板数/昨日涨停数）

**定性指标**：
- 高位股是否A杀
- 核按钮（开盘跌停）是否增多
- 天地板（涨停到跌停）是否出现
- 涨停板溢价率

## 五、短线止损纪律

### 5.1 止损铁律

```
┌─────────────────────────────────────────────────────────────────┐
│                    止损铁律                                      │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   ① 时间止损                                                    │
│      - 买入后次日不涨停，冲高卖出                               │
│      - 买入后3天内无表现，止损离场                              │
│                                                                 │
│   ② 空间止损                                                    │
│      - 单笔亏损达到-5%，无条件止损                              │
│      - 跌破关键支撑位，立即止损                                 │
│                                                                 │
│   ③ 逻辑止损                                                    │
│      - 买入逻辑被破坏，立即离场                                 │
│      - 板块退潮，个股不独立                                     │
│                                                                 │
│   ④ 情绪止损                                                    │
│      - 市场进入冰点，空仓观望                                   │
│      - 连续亏损3笔，强制休息                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 5.2 止损执行要点

1. **预设止损位**：买入前就确定止损点
2. **开盘即执行**：达到止损位，集合竞价就挂
3. **不补仓摊薄**：错了就认，不补仓
4. **止损后复盘**：分析错误原因，避免再犯

### 5.3 止盈策略

| 盈利幅度 | 操作 | 说明 |
|----------|------|------|
| +10% | 卖出50% | 锁定利润 |
| +20% | 再卖30% | 降低成本 |
| +30%以上 | 留20%或清仓 | 利润奔跑 |
| 涨停打开 | 视情况减仓 | 封不住就卖 |

## 六、小仓位管理（10%以内单笔）

### 6.1 为什么必须小仓位

**风险测算**：
- 单笔满仓：错一次亏10%，需涨11%回本
- 单笔10%：错一次亏1%，可承受多次错误
- 连续错3次：满仓亏27%，小仓位仅亏3%

**心理优势**：
- 小仓位心态更平和
- 不容易情绪化操作
- 敢于止损

### 6.2 仓位管理规则

```
┌─────────────────────────────────────────────────────────────────┐
│                    激进策略仓位管理                              │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   总资金：100%                                                  │
│                                                                 │
│   激进交易仓位：不超过30%                                       │
│   ├─ 单次交易：≤10%                                            │
│   ├─ 同时持仓：≤3只                                            │
│   └─ 分散在不同板块                                             │
│                                                                 │
│   底仓/备用：70%                                                │
│   └─ 用于稳健配置或低吸机会                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 6.3 仓位动态调整

| 市场环境 | 激进仓位 | 说明 |
|----------|----------|------|
| 情绪冰点 | 0% | 空仓观望 |
| 情绪启动 | 10% | 轻仓试错 |
| 情绪高潮 | 20-30% | 重仓龙头 |
| 情绪退潮 | 0-10% | 减仓或空仓 |

## 七、适用场景

### 7.1 适合激进搏杀的情况

1. **明确主线**：有持续性的热点题材
2. **龙头清晰**：有辨识度高的龙头
3. **情绪上升**：涨停家数增多
4. **指数配合**：大盘不拖累
5. **经验丰富**：自己有成熟的模式

### 7.2 绝对不做的情况

| 情况 | 风险 | 后果 |
|------|------|------|
| 情绪冰点 | 错杀概率高 | 连续亏损 |
| 无主线 | 随机性强 | 靠运气 |
| 高位博傻 | 接盘概率高 | 大亏 |
| 连续亏损后 | 心态失衡 | 越亏越多 |
| 自己状态差 | 判断力下降 | 错误率高 |

## 八、风险控制要点

### 8.1 最大风险

1. **A杀风险**：买入后连续跌停
2. **流动性风险**：小盘股卖不出
3. **监管风险**：特停或重点监控
4. **系统性风险**：大盘暴跌

### 8.3 交易纪律清单

**每日交易前自查**：
- [ ] 今日市场情绪如何？
- [ ] 我有明确的交易计划吗？
- [ ] 这笔交易符合我的模式吗？
- [ ] 我设置了止损位吗？
- [ ] 单笔仓位是否≤10%？
- [ ] 我状态适合交易吗？

**出现以下情况立即停止交易**：
- 当日亏损超过总资金3%
- 连续3笔交易亏损
- 情绪化交易（赌气、报复）
- 市场环境突变（大利空）

## 九、实战案例参考

### 案例1：成功一进二

**背景**：AI板块启动，某股首板突破平台

**Day1**：
- 题材：AI应用
- 首板：放量涨停，换手8%
- 封单：充足
- 判断：质量较高的首板

**Day2（一进二）**：
- 高开5%（符合预期）
- 观察5分钟，换手3%后涨停
- 打板买入（仓位10%）

**Day3**：
- 开盘涨停（加速）
- 持有观察
- 盘中开板，回封无力
- 卖出，盈利15%

**成功要素**：
- 板块强势
- 个股质量高
- 买点正确
- 及时止盈

### 案例2：失败反包

**背景**：某龙头断板，次日试图反包

**Day1**：
- 原龙头3板后断板
- 收小阳线，还能接受

**Day2**：
- 低开-3%后拉升
- 以为要反包，追高买入
- 结果冲高回落，收盘-5%

**Day3**：
- 开盘跌停
- 挂跌停卖出
- 单笔亏损15%

**教训**：
- 反包需要放量确认
- 不能提前预判
- 追高是大忌
- 这笔交易违反模式

## 十、给新手的话

**如果你符合以下任何一条，请勿使用本策略**：
- 入市不满2年
- 没有完整的交易系统
- 无法承受单笔20%以上亏损
- 容易情绪化交易
- 白天没有时间盯盘

**建议学习路径**：
1. 先学习基础技术分析
2. 建立稳健的交易系统
3. 模拟盘验证3-6个月
4. 小资金实盘（1-2万）
5. 稳定盈利后再增加资金
