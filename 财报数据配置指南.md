# 配置指南 - 财报与新闻获取

## 问题
当前akshare的财报接口不稳定，东方财富API有变更

## 解决方案

### 1. 配置Brave搜索API（获取新闻）
```bash
openclaw configure --section web
# 然后输入Brave API Key
```

获取Brave API Key：
1. 访问 https://brave.com/search/api/
2. 注册账号
3. 创建API Key（有免费额度）

配置后我可以使用 `web_search` 获取：
- 公司新闻
- 业绩预告
- 行业动态

### 2. 使用浏览器插件获取财报
可以安装 OpenClaw Browser Extension，我可以直接读取网页内容

### 3. 手动提供数据（临时方案）
你直接复制同花顺/东财的财报数据给我分析

## 当前可行的数据源
| 数据源 | 状态 | 获取方式 |
|--------|------|----------|
| 实时行情 | ✅ 正常 | API直连 |
| 涨停数据 | ✅ 正常 | API直连 |
| 财务数据 | ❌ 异常 | 需浏览器/手动 |
| 新闻资讯 | ❌ 需配置 | 配置Brave API |

## 建议
优先配置Brave API，新闻搜索对交易决策帮助最大
